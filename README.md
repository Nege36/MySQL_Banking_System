# MySQL Banking System Project

## Project Overview
This project simulates a core banking system using MySQL. It demonstrates how customer accounts, transactions, and balances are managed in a relational database environment.

## Features Implemented
- Customer account creation
- Deposits and withdrawals
- Fund transfers between accounts
- Balance checks
- Transaction logging using triggers
- Prevention of negative balances
- Large transaction monitoring

## Files in this Repository
- `schema.sql` – Database schema and table creation
- `schema.sql` – Stored procedures, triggers, and queries

## Tools Used
- MySQL
- SQL (DDL, DML, Triggers, Procedures)

## How to Use
1. Import the SQL file into MySQL
2. Run the schema to create tables
3. Execute the procedures and queries to test functionality
